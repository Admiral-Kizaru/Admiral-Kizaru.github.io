# Frontend Developer Portfolio

A GitHub Pages-ready portfolio for showcasing the projects in this workspace:

- Summarist App: React, Vite, React Router, API data, authentication modal, book pages, player, settings, and plan flow.
- Skinstric AI Analysis: React, camera and upload paths, multi-step UI, API submission, loading states, and demographic confidence results.
- Summarist Home Page: responsive HTML and CSS product landing page.

## Publish On GitHub Pages

1. Push this repository to GitHub.
2. Open the repository settings.
3. Go to Pages.
4. Choose the branch that contains this `index.html`.
5. Set the source folder to `/root`.

## Customize

Update these details in `index.html`:

- Portfolio name and headline.
- LinkedIn and email links.
- Project links after each project has a deployed URL.
